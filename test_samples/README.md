## 快速测试流程

### 单文件测试
1. 打开 SafeShrink
2. 选择 `test_samples/input/test_sensitive_docx.docx`
3. 测试"文件减肥" → 检查输出大小
4. 测试"SSD转换" → 检查表格结构
5. 测试"脱敏" → 检查敏感信息替换

### 批量测试
1. 选择 `test_samples/input/` 目录
2. 勾选"批量处理"
3. 测试不同组合：仅减肥 / 仅转SSD / 仅脱敏 / 组合
4. 检查输出目录结构

### 边界测试
- 空文件: 创建 0 字节文件测试
- 大文件: 复制多次内容创建 >10MB 文件
- 损坏文件: 用文本编辑器修改文件头
- 特殊文件名: 创建含中文、空格、特殊字符的文件
